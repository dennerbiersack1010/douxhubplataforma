# Governança de Prompts do Produto

Não existem prompts funcionais da inteligência Doux versionados nesta etapa. Quando forem criados, cada prompt deverá registrar finalidade, versão, entradas permitidas, saídas esperadas, permissões, dados sensíveis, limitações, testes e histórico de alterações.

Prompts nunca devem conter chaves, senhas, tokens, dados pessoais reais ou instruções que contornem as regras de autorização e isolamento entre clínicas.
